# linux-setup

Scripts for setting up a fresh Linux install

## Assumptions

* APT Package Manager (Debian based distro)
* May be behind corporate proxy

## Applications

* Docker
* Azure CLI 2.0
* VS Code
* Chrome
* GIMP
* Inkscape
* Git
* Node.js
* Python
* Golang
